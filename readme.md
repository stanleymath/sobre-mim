# Stanley Matheus | Portfólio Pessoal

Este repositório contém meu primeiro site pessoal, desenvolvido como parte da disciplina Desenvolvimento de Sites, ministrada pelo Prof. Dr. Fernando B. Matos. O projeto apresenta meus interesses, hobbies e áreas de atuação, com foco em um layout moderno e acessível.

---

## Sobre o Projeto

O site tem como objetivo principal reunir informações sobre mim, apresentar meu perfil profissional e destacar as áreas com as quais me identifico, como tecnologia, ciências, segurança e aprendizado contínuo.

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- Font Awesome (ícones)  
- Media queries para responsividade  
- Visual Studio Code

---

## Funcionalidades

- Seções organizadas: Sobre, Hobbies, Interesses e Contato  
- Layout responsivo para diferentes dispositivos  
- Cards interativos com ícones personalizados  
- Navegação clara com menu fixo  
- Links para LinkedIn e GitHub
